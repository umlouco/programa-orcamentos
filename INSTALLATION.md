# Installation Guide

1. Download the signed APK on the Android phone.
2. If Android asks, allow installs from the browser or file manager used for the download.
3. Open the APK and tap **Install**.
4. Open **Programa Orçamentos**.
5. Complete the company setup screen.

## Updating

Install the new signed APK over the existing app. The package id and signing key must be unchanged so Android treats it as an update and keeps local data.

Before a major update, open **Backup e restauro** and export a backup file.
