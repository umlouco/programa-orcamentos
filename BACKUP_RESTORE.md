# Backup and Restore Guide

## Export Backup

1. Open **Backup e restauro**.
2. Tap **Exportar backup**.
3. Save the `.budgetbackup` file in Downloads, Drive, OneDrive, USB storage, or another chosen folder.

## Import Backup

1. Open **Backup e restauro**.
2. Tap **Importar backup**.
3. Select a `.budgetbackup` file.
4. The app validates and restores the file.

Import replaces the current local budgets and clients with the selected backup contents.
